# 计划与进度文档说明

本目录只维护当前阶段真正需要频繁查看的计划和进度。

- `current-roadmap.md`：当前路线图与下一步
- `module-progress.md`：模块状态摘要，适合高频回写当前状态
- `page-mapping-index.md`：页面到接口映射的拆分索引
- `page-mappings-*.md`：按主题拆分的页面闭环映射
- `backlog-index.md`：阶段优先级与 backlog 的拆分索引
- `backlog-*.md`：按阶段拆分的 backlog 小文档
- `frontend-progress-index.md`：前端进度的拆分索引
- `frontend-progress-*.md`：按主题拆分的前端进度小结
- `todo-index.md`：执行 Todo 的主题入口与拆分索引
- `todo-*.md`：按主题拆分的 Todo 子文档
- `项目进度文档.md`：面向后来者接手的项目状态总览与阅读顺序
- `生产环境并发与性能治理计划.md`：面向生产环境高并发、长任务、媒体流量和容量治理的专项计划
- `生成资产前端访问体验技术路线方案.md`：针对大体积生成资产的预览、播放、下载与分辨率体验制定的媒体交付路线方案
- `生成资产前端访问体验方案A-轻量媒体分层交付.md`：当前阶段优先落地的轻量媒体分层交付方案
- `生成资产前端访问体验方案B-生产级媒体资产管线升级.md`：中期升级到对象存储、CDN 与转码切片的目标方案
- `开发总结文档.md`：工程经验、排障结论与长期有效做法
- `总结成功经验.md`：沉淀已在真实云端排障中验证过的成功案例、判断顺序与可复用命令
- `功能模块Todo文档.md`：Todo 层的跨模块执行母表、阶段摘要与统一回写规则
- `archive-index.md`：历史大文件索引

详细逐日变更统一查看根级 `CHANGELOG.md`。
