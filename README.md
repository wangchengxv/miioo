# 🌟 Miioo AI - 短剧生产流程化工具平台

Miioo AI 是面向短剧生产的流程化工具平台，围绕“剧本 → 主体 → 分镜 → 出片”构建完整闭环。平台功能完全免费，只需接入您自己的 API key，即可开启漫剧制作之旅。

---

## 🌟 核心功能

### ✅ 创作模式
支持图片生成、视频生成、音频生成等创意工作，生成内容保存在云端资产库，支持在项目中复用。

### ✅ 项目模式
适合短剧项目的流程化制作，支持生成和导入剧本、自动抽取主体、智能处理分镜。

### ✅ 资产库
项目资产和创作资产独立存储，可一键下载完整项目文件，各资产支持任意复用，一键参考。

### ✅ API key 接入
平台功能全免费，无复杂的充值积分兑换机制，您无需在我们平台付费，接入任意第三方的 API key 即可调用服务。

### ✅ 安全合规
平台采用前端加密技术保存您的 API key，无盗用风险。
