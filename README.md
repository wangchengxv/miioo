# miioo

AIGC 影视化工作流产品，面向影视创作者，通过 AI 辅助完成从剧本创作到成片剪辑的全流程工作。

## 快速上手

```bash
git clone https://github.com/wangchengxv/miioo.git
cd miioo
npm install
npm run dev   # http://localhost:5173
```
